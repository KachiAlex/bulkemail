## 📋 Quick Setup (5 Minutes)

### 1. Create Icons (2 minutes)

**Easiest way:**
1. Open `create-icons.html` in Chrome
2. Click "Download All Icons"
3. Move files to `icons/` folder

**Or use online:**
- Go to: https://favicon.io/favicon-generator/
- Enter "CRM" or "🔍"
- Color: `#0077b5`
- Download and rename files

### 2. Load Extension (1 minute)

1. Open: `chrome://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select: `browser-extension` folder

### 3. Test (2 minutes)

1. Go to LinkedIn profile
2. Click extension icon
3. Click "Log In" (first time)
4. Click "Extract Profile Data"
5. Click "Share to Database"
