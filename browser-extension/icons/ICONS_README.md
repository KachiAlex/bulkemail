# Extension Icons

This folder should contain the following icon files:

- `icon16.png` - 16x16 pixels (for browser toolbar)
- `icon48.png` - 48x48 pixels (for extension management page)
- `icon128.png` - 128x128 pixels (for Chrome Web Store)

## Creating Icons

You can create icons using:
1. Online icon generators (search for "chrome extension icon generator")
2. Image editing software (Photoshop, GIMP, etc.)
3. Online tools like:
   - https://www.favicon-generator.org/
   - https://favicon.io/
   - https://icon.kitchen/

## Icon Design Tips

- Use a simple, recognizable symbol (e.g., 🔍, 📊, 💼)
- Use brand colors if applicable (e.g., LinkedIn blue: #0077b5)
- Ensure icons are clear at small sizes (16x16)
- Use transparent background (PNG format)

## Temporary Solution

For development/testing, you can:
1. Use a placeholder icon (any 16x16, 48x48, 128x128 PNG)
2. Create a simple colored square with text
3. Use an online generator to create all three sizes from one image

## Quick Icon Creation

You can create a simple icon using any online tool:
1. Go to https://favicon.io/favicon-generator/
2. Enter text "CRM" or "🔍"
3. Choose colors
4. Download and extract the PNG files
5. Rename and place in this folder:
   - favicon-16x16.png → icon16.png
   - favicon-32x32.png → icon48.png (resize to 48x48)
   - favicon-96x96.png → icon128.png (resize to 128x128)
